Les formulaires


Pour récupérer les données d'une requête POST : $_POST ou GET : $_GET
qui vont remplacer les "getParameter" vus en JEE


Une autre option possible consiste à combiner en un seul fichier le php et le html.
Toutefois, il est impératif de vérifier un if(isset($_POST)) au préalable
avant d'en gérer l'affichage.

Des méthodes permettant de filtrer les balises : filter_var et filter_input.
Un exemple de syntaxe :
$email = "olivier@guilben.bzh";

if (filter_var($email, FILTER_VALIDATE_EMAIL)) {
echo("$email is a valid email address");
} else {
echo("$email is not a valid email address");
}
Il est possible de récupérer plusieurs valeurs en même temps avec les variants filter_var_array

En utilisant une appellation différente de name="nom[key]" on pourra passer des valeurs en
tableau directement
